Dies ist die Elementverwaltung, in der Elemente erstellt und verändert werden. Elemente sind Instanzen von Elementtypen und sind die Bausteine aus denen Beitrag- und Mediendokumenten. Sie sollten zunächst Bric::ElementAdmin lesen um zu verstehen, wie Sie Ihre Elementhierarchie planen sollten.

Suche
Sucht Elemente nach Namen. Eine leere Suche listet alle Elemente auf, über die Buchstaben erhäten Sie eine alphabetische Liste. Sie können auch auch über das 'Typ auswählen' Menü einen Elementtyp zum auflisten vorgeben.
Neues Element hinzufügen
Klicken Sie hier um ein neues Element anzulegen. Sie werden zur Elementprofilseite weitergeleitet.
Vorhandene Elemente
Klicken Sie Edit um das Profil eines Elementes zu ändern. Klicken Sie Protokoll um die Geschichte der Ereignisse anzusehen. Um ein Element zu löschen, setzen Sie einen Haken in die Löschen-Box, gehen Sie ans Ende der Seite, und klicken Sie auf den Knopf Gewählte Löschen.
