This is the Element Manager, where you create and edit Elements. Elements are instances of Element Types, and are the building blocks of story and media documents. You should first read Bric::ElementAdmin in order to understand how to plan your element hierarchy.

Search
Searches Elements by name. Do a blank search to list all Categories or click a letter for an alphabetical list. You can also select a type of element to list using the 'Pick a Type' menu.
Add a New Element
Click this link to create a new Element. It brings you to the Element Profile page.
Existing Elements
Click Edit to modify an Element's profile. Click Log to view its event history. To delete an Element, check the Delete box, scroll to the bottom of the page and click the Delete Checked button.
