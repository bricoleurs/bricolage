Здесь вы создаете и редактируете Типы Элементов. Типы Элементов предназначены для группирования Элементов по конкретным свойствам.

Поиск (Search)
Поиск Типов Элементов по имени. Для получения списка всех Типов Элементов оставьте поле поиска пустым. Для просмотра списка Типов Элементов в алфавитном порядке нажмите на нужную букву.
Добавить новый Тип Элементов (Add a New Element Type)
Эта ссылка используется для создания нового Типа Элементов. По ней выполняется переход на страничку «Профиль Типа Элементов».
Существующие Типы Элементов (Existing Element Types)
Нажмите Редактировать (Edit) для модификации профиля Типа Элемента. Нажмите Журнал (Log) для просмотра списка всех действий, выполненных с Типом Элемента. Для удаления Типа Элемента поставьте отметку в окошке Удалить (Delete), а затем нажмите клавишу Удалить отмеченные (Delete Checked) в конце страницы.
